# Real Time User Activity Analysis

This project provides a real-time analysis pipeline for user activity in an e-commerce application. It utilizes Python for data processing, Docker for containerization, Logstash for log ingestion, and Kibana for dashboard visualization.

## Project Overview

The goal of this project is to process, analyze, and visualize user activity data generated by an e-commerce application. The solution is designed for scalability and real-time insights, making it suitable for production environments.

### Key Components

- **Ecommerce_app_analysis.py**:  
  Processes and analyzes raw user activity data from the e-commerce platform. It extracts key metrics such as user sessions, product clicks, purchases, and behavioral patterns. This script can be extended for more complex analytics as required.

- **docker-compose.yml**:  
  Defines the services required for the analysis pipeline, including Elasticsearch, Logstash, and Kibana (ELK stack). This allows for easy deployment and orchestration of the entire analytics infrastructure.

- **logstash.conf**:  
  Configuration file for Logstash, specifying how log data should be ingested, filtered, and sent to Elasticsearch. This file is crucial for parsing the raw logs into structured data.

- **Kibana Dashboard**:  
  A customizable dashboard for visualizing key metrics and trends in user activity. Kibana provides interactive charts and real-time exploration of the ingested data.
